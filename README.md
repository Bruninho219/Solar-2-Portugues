# Solar-2-Portugu-s
Tradução para português brasileiro do game Solar 2

O diretório pode variar de acordo com o locar de instalação.

Local: X:\SteamLibrary\steamapps\common\Solar 2\Content\Languages

Instalação:
1- Vá ao local de instalação, conforme dito acima
2- Faça um backup de seguarança do arquivo strung.en.txt para strung.en-bkp.txt por exemplo
3- Faça o download do arquivo strung.en.txt e cole no local
