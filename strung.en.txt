// Contém todos os textos do jogo no formato:
// nome : valor 
// Comentários são definidos por linhas que começam com //

// Qualquer coisa como {0} ou {1} representa texto definido no código e deve ser mantido.

// Este jogo pode ser traduzido para outros idiomas baseados em latim ou cirílico (desculpe, idiomas asiáticos não são suportados no momento)
// Você pode testar se o jogo suporta o idioma colocando o alfabeto completo no item "splash_pc". Se algum caractere aparecer como ?, ele não é suportado.
// Para usar outro idioma no jogo, copie este arquivo e traduza as partes de 'valor' (lado direito).
// Nomeie o arquivo como "strung.[codigo_idioma].txt"    por exemplo, francês seria "strung.fr.txt"
// Se o Windows estiver configurado com esse idioma padrão, o jogo usará automaticamente.
// Coloque o arquivo na pasta "Languages" no diretório de instalação.
// Ou dentro da pasta de salvamento do jogo (onde fica o arquivo "config"), crie uma pasta "Languages" e coloque o arquivo lá.
// A pasta de salvamento geralmente fica em "Documents/My Games/Solar2". Procure por "config" no sistema após rodar o jogo pelo menos uma vez.
// Além disso, para o jogo detectar, vá no arquivo config e na opção "AvailableLanguages" adicione os códigos separados por vírgula, exemplo:
// AvailableLanguages=en,es,fr
// Se configurado corretamente, o idioma aparecerá nas opções.
